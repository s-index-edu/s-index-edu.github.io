Set up analytics

This site uses GoatCounter — a free, open-source, cookie-free analytics service. Follow these steps to activate statistics:

  1. Create a free account at goatcounter.com.
  2. Your site code will be something like your-site-name (gives you your-site-name.goatcounter.com).
  3. In every HTML page, add this snippet just before </body>: 
    <script data-goatcounter="https://YOUR-CODE.goatcounter.com/count"
        async src="//gc.zgo.at/count.js"></script>
  4. To enable the statistics on this page, open stats.html and set the GOATCOUNTER_CODE constant at the top of the page script to your GoatCounter site code.
  5. In your GoatCounter settings, enable "Allow public access to stats" so the stats dashboard can be embedded here.

GoatCounter does not collect personal data or use cookies, so no cookie consent banner is required. It is GDPR, CCPA, and PECR compliant.
